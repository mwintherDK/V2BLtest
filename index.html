Liste med anvendte Git commands

  • Git clone - cloner dit remote repo til din computer og merger evt.
  • Git add [-A] - tilføjer alle filer og ændringer til stage/index/venteområde
  • Git commit -m "Indsæt RELEVANT besked her"
  • Git push - skubber dit lokale repo ud i skyen
  • Git status tjekker status på dit LOKALE repo/venterum/working branch
